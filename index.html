Music M (VK M)

WinUI3 C# VK API Pet Project

💡 Концепция

Music M - это не просто pet-проект, а страсть и энтузиазм, воплощённые в реальность. Приложение решает проблему громоздкого процесса прослушивания музыки ВКонтакте, предлагая быстрый и удобный доступ к вашей медиатеке.

🚀 Разработка

Изначально задуманное как простой плеер, приложение выросло в полноценный продукт на WinUI3 и C#. Использование VKNet API обеспечивает стабильную работу и прямой доступ к музыкальной библиотеке ВК.

✨ Особенности

  • Быстрый доступ без браузера
  • Чистый и минималистичный интерфейс
  • Высокая производительность
  • Интеграция с VK музыкой

Android Development

📱 Native ☕ Java 📦 APK 🍎 iOS Port

Приложение для "Здоровеево"

Полноценное нативное приложение для интернет-магазина, разработанное с нуля как дипломный проект.

Вызовы разработки

Пришлось начинать с нуля без исходников предыдущего приложения. За 4 месяца создал стабильное приложение с корзиной, избранным, новостями и исправил редкие краши на старых устройствах.

Минимизация и обусфакция

Сейчас, уже все привыкли, что мобильные приложения весят больше 100 МБ. Но мне не нравится подобная тенденция. И я стараюсь уменьшить размер своих приложений. Таким образом, в процессе исследования возможностей обусфакции приложений под Android, получилось достичь значительного прогресса. Ранее приложение весило около 80 мб. Получилось достичь 2,4 МБ, не потеряв кода.
